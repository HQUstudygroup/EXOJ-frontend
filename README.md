# EXOJ 拖拽式表格处理

## Quick Start

#### 1. 下载环境

安装 NodeJS： [https://nodejs.org/en/download](https://)

安装完成并配置环境变量后

```bash
npm install -g pnpm@latest-10 # 安装 pnpm 依赖
```

#### 2. 克隆项目和安装项目依赖

```bash
git clone https://github.com/HQUstudygroup/EXOJ-frontend.git # 克隆项目

cd project-name # 到项目目录下

pnpm i # 安装项目依赖
```

#### 3. 启动项目

**开发模式：**

```bash
pnpm dev
```

**生产模式：**

```bash
pnpm build
pnpm start
```
